# Jun Seok Kim - Research Portfolio Website

A modern, production-ready personal research and developer website showcasing the transition from combustion engineering research to AI/Computer Vision and scientific document intelligence.

## Overview

This website presents Jun Seok Kim as a researcher/developer with expertise spanning:
- Hydrogen-enriched combustion and flame dynamics
- Scientific document intelligence and evidence extraction
- Computer vision for scientific figures
- AI-assisted software development
- Product-oriented prototyping

## Target Audience

- Startup CTOs and CEOs
- AI/ML engineers and researchers
- Computer vision researchers
- Scientific AI researchers
- Engineering R&D teams
- Graduate admissions reviewers
- Technical recruiters
- Overseas professors

## Core Positioning

**Combustion Researcher → Scientific AI / Computer Vision Engineer**

Primary tagline: *Combustion researcher building Scientific AI systems for engineering evidence.*

## Technology Stack

- **Frontend Framework**: React 18.3.1
- **Styling**: Tailwind CSS 4.1
- **UI Components**: Custom components built with Radix UI primitives
- **Icons**: Lucide React
- **Animation**: Motion (Framer Motion)
- **Build Tool**: Vite 6.3.5
- **Package Manager**: pnpm

## Development

This project uses Vite and is configured for the Figma Make environment.

### Important Notes

- Do NOT run `vite build` or `npm run build` in this environment
- Do NOT create `index.html` - the entrypoint is auto-generated at runtime
- The Vite dev server is already running
- Do NOT start it manually with `npm run dev` or `vite`

### Available Scripts

All scripts should be run via pnpm:

```bash
pnpm install        # Install dependencies (if needed)
```

## Design Guidelines

### Typography
- Large, bold headlines for section titles
- Comfortable reading sizes for body text

### Color Usage
- Accent color (#00ff9f) used sparingly for emphasis

- Rounded corners (0.75rem radius)
- Subtle border styling

- Consistent vertical rhythm
- Generous padding in cards
- Clear section separation
- Responsive grid layouts
